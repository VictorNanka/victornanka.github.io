Command Line Heros 播客稿翻译计划

Command Line Heros 是 Redhat 推出的原创播客，目的是讲述技术知识和计算机史，采访了不少信息革命的见证者。

目前博客已经出了四季，每季大概8个episode，质量十分不错。

计划以一周三到四篇的更新频率翻译该博客的稿子，以飨读者。

悉知，本人纯粹出于个人学习技术和锻炼英语能力的动机翻译此作品，仅为学习和交流，该翻译没有获得 Redhat 授权。
